Passi da seguire per creare l'ambiente di sviluppo misurainternet-speedtest
1- installare
	1.1- git
	1.2- nodejs (con npm)
2- clonare il progetto da github con 'git clone'
3- nella root del progetto lanciare il comando 'npm init' e lasciare tutti
   valori di default per inizializzare il file 'package.json'
4- lanciare il comando 'npm install ws'
